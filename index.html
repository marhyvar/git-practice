Configuring Git

After installing Git, you should configure at least these:

$ git config --global user.name "Your Name"

$ git config --global user.email youremailaddress@example.com

These are included in every Git commit you make.

If you do not want to use Git's default editor, you should configure your own preferred editor:

$ git config --global core.editor editor-of-choice

Here's how you can check your settings:

$ git config --list
